# 中文技术文档写作规范

## 核心原则
排版服务于阅读体验，规范服务于一致性。

## 空格规则
- 中英文之间加空格：`使用 Redis 进行版本管理`
- 中文与数字之间加空格：`包含 3 个新功能`
- 链接前后加空格：`请参考 [文档](url) 获取信息`

## 标点符号
- 中文语境使用全角标点：`注意：该接口需要鉴权`
- 英文/代码使用半角标点：`Run npm install, then test`

## 术语处理
**保留英文：** React, API, SDK, CLI, ORM, CI/CD
**翻译为中文：** 数据库、服务器、浏览器、框架
**首次出现标注翻译：** 消息队列（Message Queue）实现异步通信
