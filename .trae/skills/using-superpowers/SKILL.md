# 使用 Superpowers

## 核心规则
在任何响应或操作之前调用相关或被请求的技能。哪怕只有 1% 的可能性某个技能适用，都应该调用该技能来检查。

## 技能优先级
1. **流程技能优先**（头脑风暴、调试）— 决定如何处理任务
2. **实现技能其次**（前端设计、mcp-builder）— 指导执行

## 中国特色技能路由
| 场景 | 调用技能 |
|------|---------|
| 代码审查且中文沟通 | chinese-code-review |
| 使用 Gitee/Coding/极狐 | chinese-git-workflow |
| 编写中文技术文档 | chinese-documentation |
| 编写 git commit message | chinese-commit-conventions |
| 构建 MCP 服务器 | mcp-builder |

## 如何访问技能
使用 Skill 工具加载技能内容。绝不要用 Read 工具读取技能文件。
