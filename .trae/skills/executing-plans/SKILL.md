# 执行计划

加载计划，批判性审查，执行所有任务，完成后报告。

**开始时宣布：** "我正在使用 executing-plans 技能来实现此计划。"

## 流程

### 步骤 1：加载并审查计划
1. 读取计划文件
2. 批判性审查——识别问题或疑虑
3. 有疑虑则向用户提出
4. 无疑虑则创建 TodoWrite 并继续

### 步骤 2：执行任务
对每个任务：
1. 标记为进行中
2. 理解目标
3. 严格按计划步骤执行
4. 运行验证
5. 提交变更
6. 标记为已完成

**批量审查检查点：** 每 3 个任务后暂停回顾

### 步骤 3：处理异常
- 测试失败 → 定位原因，修复
- 依赖缺失 → 停下来报告
- 指令不清 → 不要猜测，询问用户

### 步骤 4：完成开发
所有任务完成并验证后：
- 宣布："我正在使用 finishing-a-development-branch 技能来完成此工作。"
- 使用 superpowers:finishing-a-development-branch 处理合并/PR
